📞 TROUBLESHOOTING
================================================================================

Q: Flask app won't start
A: Check if port 5000 is busy
   $ lsof -i :5000
   Try: python app.py --port 5001

Q: "ANTHROPIC_API_KEY not found"
A: Key is missing from .env file
   Check: cat .env | grep ANTHROPIC
   Fix: Add key to .env and restart

Q: Shopify still returning 403
A: Credentials might be in wrong format
   Verify: API credentials are space-trimmed in .env
   Try: Regenerate credentials from Shopify again

Q: GA4 shows "Not authenticated"
A: OAuth hasn't been completed
   Fix: Ask agent a question → Click OAuth link → Grant access
   Note: One-time setup per installation

Q: Agent doesn't use vault data
A: Vault path might be incorrect
   Check: VAULT_PATH in .env points to correct location
   Test: Print files in that location
